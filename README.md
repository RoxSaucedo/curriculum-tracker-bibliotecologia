# 📚 Rastreador de Plan de Estudios - Bibliotecología

Aplicación web interactiva para estudiantes de la Tecnicatura Superior en Bibliotecología.

## ✨ Características

- ✅ Rastreo visual del progreso académico
- ✅ Sistema inteligente de correlativas
- ✅ Colores institucionales IFTS Nº 13
- ✅ Modo oscuro/claro
- ✅ Responsive (mobile + desktop)
- ✅ Guardado automático en navegador
- ✅ Sin servidor necesario

## 🚀 Uso

1. Abre `curriculum-tracker-verde.html` en tu navegador
2. Haz clic en las materias para cambiar su estado
3. Tu progreso se guarda automáticamente

## 📊 Estados

- 🔒 **Bloqueada** - Requisitos sin aprobar
- ○ **Pendiente** - No iniciada
- 📖 **Cursando** - En progreso
- ✓ **Regularizada** - Cursada, sin final
- ✓ **Aprobada** - Completada
